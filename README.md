# Cross-Correlation and FFT Phase Analysis for Defect Detection in Active Thermography of FRP Composites

Research project conducted at the **InfraRed Imaging Laboratory (IRIL), SeNSE, IIT Delhi** under the supervision of **Dr. Ravibabu Mulaveesala**.

A per-pixel signal processing pipeline for detecting subsurface defects in Carbon and Glass Fibre-Reinforced Polymer (CFRP/GFRP) composites using active thermography. The pipeline applies linear detrending, full cross-correlation against a sound reference pixel, main lobe extraction, and FFT phase computation to thermal video data.

---

## Dependencies

Install the following Python packages before running any scripts:

```bash
pip install numpy scipy matplotlib opencv-python
```

| Package | Usage |
|---|---|
| `numpy` | Array operations, FFT, cross-correlation |
| `scipy` | Linear regression for detrending (`scipy.stats.linregress`) |
| `matplotlib` | Plotting pixel intensity and correlation graphs |
| `opencv-python` | Video I/O, greyscale conversion, normalisation |

Python 3.8 or later is recommended.

## Source Code — `src/`

Scripts are organised into three subfolders, one per dataset. Each folder contains an independent, self-contained pipeline that reads from and writes to its corresponding `data/vX/` folder.

### `src/v1_cfrp_lfm/` — CFRP with LFM Excitation

| File | Description |
|---|---|
| `cfrp_lfm_pipeline.py` | Main interactive analysis script for the CFRP LFM dataset. Runs the full pipeline: greyscale normalisation → linear detrending → cross-correlation → main lobe extraction → FFT phase. Results can be displayed, saved as video, or plotted via a menu. |
| `display_video.py` | Utility script that opens the raw CFRP LFM video and plays it back as greyscale frames. Used for a quick visual check of the input data. |
| `test_numpy_correlate.py` | Scratch script verifying the behaviour of `numpy.correlate` on toy arrays. Not part of the research pipeline. |

### `src/v2_gfrp_ap/` — GFRP with Active Pulsed Excitation

| File | Description |
|---|---|
| `gfrp_ap_pipeline.py` | Main analysis script for the GFRP Active Pulsed dataset. Extends the v1 pipeline with full-mode cross-correlation, discretisation (zero-padding to variable levels), and main lobe cropping. Includes a function to extract and save the correlation main lobe as a separate video. |
| `test_correlate_size.py` | Scratch script that prints the output length of `numpy.correlate` in full mode. Not part of the research pipeline. |

### `src/v3_gfrp_2000/` — GFRP with Flux Excitation (2000 frames)

| File | Description |
|---|---|
| `gfrp_flux_pipeline.py` | Main analysis script for the 2000-frame GFRP flux dataset. Applies the same pipeline as v2 with fixed 160-level discretisation. Contains stub functions (`graph2`, `graph3`) for planned future analysis of FFT phase vs. defect depth. |

## Running the Scripts

Each pipeline script loads its detrended and correlation matrices from cached `.pkl` files on startup. If no cache exists, it recomputes and saves them automatically (this is slow — expect several minutes for large datasets).

Run from the project root:

```bash
python src/v1_cfrp_lfm/cfrp_lfm_pipeline.py
python src/v2_gfrp_ap/gfrp_ap_pipeline.py
python src/v3_gfrp_2000/gfrp_flux_pipeline.py
```

Each script presents an interactive menu with options to display, save, or plot the output at each pipeline stage.
